Aide

Catégorie

  • Les catégories sont des groupes de budgets. Ex: Maison et Loisirs.

Budget

  • Les budgets sont des groupes de dépenses. Ex: Voiture (révision, contrôle technique, essence, etc.)

Transaction

  • Il est impossible de supprimer une transaction car le but est de tout enregistrer. Si il est nécessaire de supprimer une dépense, vous pouvez créer une nouvelle transaction avec le montant opposé (5€ -> -5€).

Transactions périodiques

  • Idem par rapport aux transaction avec un côté récurrent (l'éléctricité se paye en général tous les mois, etc.).
